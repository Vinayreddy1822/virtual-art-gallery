# Virtual Art Gallery

Grand Gallery is a responsive virtual art museum website where users can explore curated collections, exhibitions, and digital artworks.


## Features

- Browse 49 curated artworks
- Filter artworks by category (Nature, Photography, Digital Art, Art Work)
- Live search with instant results
- 3D themed exhibition galleries
- Bookmark artworks using LocalStorage
- Artwork preview modal popup


## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
- Lucide Icons
